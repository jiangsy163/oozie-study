The Oozie Web Services API is a HTTP REST JSON API.响应为UTF-8 格式。
支持的终端为：

    /versions
    /v1/admin
    /v1/job
    /v1/jobs
    /v2/job
    /v2/jobs
    /v2/admin
    /v2/sla
